# JavaScript Calculator

This is a **simple calculator application** built using HTML, CSS, and JavaScript. The calculator supports basic arithmetic operations including addition, subtraction, multiplication, and division, with real-time display updates and user input handling.

---

## Project Overview

The calculator allows users to:

- Input numbers via on-screen buttons  
- Perform basic arithmetic operations (`+`, `-`, `*`, `/`)  
- Handle decimal numbers correctly  
- Display the current operation and result dynamically  
- Reset all inputs using the clear button  

This project demonstrates practical skills in building interactive web applications using JavaScript and manipulating the DOM.

---

## Features

- Clickable number buttons for input  
- Operator buttons for arithmetic operations  
- Decimal input support  
- Clear (AC) button to reset the calculator  
- Real-time display of current input and previous operation  
- Automatic evaluation when pressing the equals button  

---

## Skills Demonstrated

- JavaScript functions for arithmetic operations  
- Event handling with `addEventListener`  
- DOM manipulation to update displays dynamically  
- Handling user input, including decimals and operator precedence  
- State management for interactive applications  

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6+)
